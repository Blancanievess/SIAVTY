About Us

We are students in the 3rd year of Geomatics and Topographic Engineering at the Hassan II Agronomic and Veterinary Institute. This project is part of our training in Location-Based Services (LBS) and Internet of Things (IoT) and aims to develop an intelligent security solution.

SIAVTY - GeoSecure Portal

Our solution integrates the latest advancements in IoT, LBS, and facial recognition to provide advanced access control and real-time surveillance. This security geoportal is built around two main features:

  • Facial Recognition for Access Control: Only pre-approved individuals can enter restricted areas.
  • Real-Time Security Agent Tracking: Active monitoring and task assignment through geofencing.
  • Instant Alerts & Intrusion Management:An automated system to detect and report unauthorized access attempts.

We believe in intelligent automation to enhance security and ensure effective monitoring of sensitive environments.
